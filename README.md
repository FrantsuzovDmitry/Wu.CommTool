# Wu通讯调试工具

## 功能介绍
1. Modbus Rtu 
- 自定义帧  支持自动校验
- 搜索设备  可设置多种组合，自动搜索设备
- 数据监控  可设置字节序、数据类型等，支持读取与写入
- 自动应答  
2. Mqtt Server
- 可右键对Json格式的数据进行格式化查看
3. Mqtt Client
- 可右键对Json格式的数据进行格式化查看
4. Json格式化查看
